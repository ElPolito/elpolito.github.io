# Up Proximity

Up Proximity est un jeu mobile en deux dimensions dans lequel le joueur contrôle une fusée qui doit rattraper une soucoup volante en évitant des astéroides.

Le développement du jeu a commencé en 2016 et un première version est sortie assez rapidement, cependant, les graphismes n'étaient pas du tout cohérent (car tous récupérés à différents endroits sur internet). Le projet a donc été abandonné pendant un petit moment.

En 2019, j'ai décidé de reprendre le projet en partant plus ou moins de zero tout en récupérant certains bout de code du projet initial. J'ai cette fois ci pris le parti de réaliser tout les designs moi même sur Inkscape. Le résultat est bien plus joli que la première fois et le nouveau jeu est sorti sur le PlayStore peu de temps après.
